# Installation
## 0. Clone the repository and cd into it

```bash
git clone https://github.com/Kuuro-neko/Firefox-custom-css.git
cd Firefox-custom-css
```

## 1. Enable userChrome.css

Go to `about:config`

Set `toolkit.legacyUserProfileCustomizations.stylesheets` to `true`

## 2. Windows

Run **as an administrator** `windows.bat`

## 2. Linux

Run `linux.sh`

# Edit

You need to rerun the script only if you added new files (it's all symlinks)
